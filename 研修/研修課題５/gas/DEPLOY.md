# GASデプロイ手順

## 1. GASエディタを開く

1. スプレッドシートを開く
   - URL: https://docs.google.com/spreadsheets/d/1Jm8BvzlBH-WG64f6Fzr2GMsivOcag_lFuk-6G25BicA/edit

2. 「拡張機能」→「Apps Script」をクリック

## 2. Webアプリとしてデプロイ

1. GASエディタで「デプロイ」→「新しいデプロイ」をクリック

2. 「種類の選択」で「ウェブアプリ」を選択

3. 設定を入力：
   - **説明**: 勤怠管理アプリ v1
   - **次のユーザーとして実行**: 自分
   - **アクセスできるユーザー**: **全員**（重要！）

4. 「デプロイ」ボタンをクリック

5. 権限の承認を求められたら：
   - 「アクセスを承認」をクリック
   - Googleアカウントを選択
   - 「詳細」→「勤怠管理アプリ（安全ではないページ）に移動」をクリック
   - 「許可」をクリック

6. **デプロイURL**をコピー
   - 形式: `https://script.google.com/macros/s/XXXXX/exec`
   - このURLをメモしておく

## 3. フロントエンドに設定

1. ブラウザで http://localhost:8000 を開く

2. 初期設定モーダルで以下を入力：
   - **ユーザー名**: あなたの名前
   - **GAS WebアプリURL**: 上記でコピーしたデプロイURL

3. 「設定を保存」をクリック

## 4. テスト

1. 「出勤」ボタンをクリック

2. スプレッドシートを確認
   - 新しい行が追加されているはず
   - 日付、ユーザー名、出勤時刻、ステータスが記録されている

## トラブルシューティング

### エラー: "Script function not found: doPost"
- GASコードが正しくデプロイされていない
- GASエディタでコードを確認し、再度デプロイ

### エラー: "Authorization required"
- 「アクセスできるユーザー」が「全員」になっているか確認
- 再度デプロイして新しいURLを取得

### スプレッドシートに記録されない
- GASのログを確認（GASエディタで「実行」→「実行ログ」）
- スプレッドシートIDが正しいか確認
- シート名が「勤怠記録」になっているか確認

## 次のステップ

デプロイが完了したら、以下を確認：
- [ ] デプロイURLを取得
- [ ] フロントエンドに設定
- [ ] 出勤ボタンをクリック
- [ ] スプレッドシートに記録されることを確認
