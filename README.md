# 勤怠管理アプリ セットアップ手順

自動連携（Clasp）がうまくいかなかったため、以下の手順で手動セットアップをお願いします。

## 1. GASプロジェクトの作成
1. [Google Apps Script](https://script.google.com/) にアクセスし、「新しいプロジェクト」を作成します。
2. プロジェクト名を「勤怠管理アプリ」などに変更します。

## 2. コードのコピー
以下のファイルの内容を、GASエディタの該当ファイルにコピー＆ペーストしてください。

- **Code.js**: `コード.js` (または `Code.gs`) に貼り付け。
- **index.html**: ファイルを追加（「+」ボタン -> HTML）し、名前を `index` にして貼り付け。
- **appsscript.json**: 「プロジェクトの設定」で「「appsscript.json」マニフェスト ファイルをエディタで表示する」にチェックを入れ、表示されたファイルに貼り付け。

## 3. スクリプトプロパティの設定
1. 左側の「プロジェクトの設定」（歯車アイコン）をクリック。
2. 「スクリプト プロパティ」の「スクリプト プロパティを追加」をクリック。
3. プロパティ: `LINE_TOKEN`
4. 値: (あなたのLINE Notifyトークン)
5. 「スクリプト プロパティを保存」をクリック。

## 4. デプロイ
1. 右上の「デプロイ」 -> 「新しいデプロイ」をクリック。
2. 種類の選択（歯車） -> 「ウェブアプリ」。
3. 説明: 「初版」など。
4. 次のユーザーとして実行: 「自分」。
5. アクセスできるユーザー: 「自分のみ」または「全員」（用途に合わせて）。
6. 「デプロイ」をクリック。
7. 発行された「ウェブアプリのURL」をコピーし、スマホで開いてホーム画面に追加してください。
